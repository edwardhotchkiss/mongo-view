## Installation

```bash
$ [sudo] npm install mongo-view -g
```

## Running Mongo-View (http://localhost:9000/)

```bash
$ mongoview
```

## Running Mongo-View (http://localhost:9001/)

```bash
$ mongoview run --port 9001
```
